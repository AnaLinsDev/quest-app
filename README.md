# quest-app

## Tecnologias:
- NuxtJs
- Vuetify
- Vue
- Vuex
- SASS

## Como subir a aplicação:
- yarn install
- yarn dev

## Como testar:
- yarn install
- yarn test

## Subir o json-server: (Versão inicial do projeto)
- Entre no terminal na pasta data
- json-server --watch db.json --port 5000
